# My-vim-config
My personal vim configuration

### Quick auto installation
###### With https:
`git clone https://github.com/tjespe/My-vim-config ~/.vim && ./install.sh`
###### With ssh:
`git clone ssh://git@github.com/tjespe/My-vim-config ~/.vim && ./install.sh`

### Quick updating
`~/.vim/install.sh`
